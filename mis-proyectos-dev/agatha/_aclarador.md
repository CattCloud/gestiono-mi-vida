# 🧠 Agatha

## Aclarador

Agatha es mi cabina de mando personal — una entidad unificada formada 
por datos (los `.md` del vault), vistas (Kanban, calendario, generador 
de horario, etc.) y una voz IA con la que colaboro a diario.

No es un tablero, no es una app, no es un asistente: es las tres cosas 
juntas, indistinguibles. Cuando le hablo, le hablo a Agatha. Cuando 
abro su tablero, abro a Agatha. Cuando ella me responde, responde 
desde su propia memoria —que es mi vault.

La primera vista que se construye es un **Kanban propio** (HTML + CSS 
+ JS dentro de Obsidian, vía Dataview JS u otra tecnología por 
definir) que reemplaza al plugin CardBoard. Después se incorporarán 
otras vistas y herramientas que comparten el mismo cerebro de datos.

Visión a largo plazo: Agatha es el prototipo de una futura app web 
propia. Lo que se diseñe aquí debe poder migrarse después.

Estado actual: en planeación. La sesión del 2026-04-25 abrió la 
misión, definió la visión de producto, y dejó armado el documento vivo 
`vision-producto.md` que recopila decisiones de negocio. Las 
siguientes sesiones aterrizarán anti-objetivos, glosario, casos de uso 
y features antes de pasar a Claude Code para implementación.
