# Mis Proyectos Dev

Aquí viven mis proyectos de software personales — lo que construyo por cuenta propia, por curiosidad, por aprendizaje o por necesidad.

Cada proyecto es una **misión**. No importa si es grande o pequeño, si está activo o pausado. Si es código mío y no es del trabajo, va aquí.

## Qué tipo de misiones van aquí

- Apps o herramientas que desarrollo para mí
- Proyectos experimentales o de aprendizaje
- Contribuciones a proyectos open source
- Ideas de software que quiero explorar
