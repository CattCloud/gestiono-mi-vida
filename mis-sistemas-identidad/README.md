# Mis Sistemas de Identidad

Aquí viven los hábitos y sistemas que estoy construyendo para mi crecimiento personal. No son tareas sueltas — son procesos que estoy integrando a mi vida.

Un "sistema" es algo que repito con intención para convertirme en la persona que quiero ser. Cada sistema es una **misión** con su propio aclarador y tareas.

## Qué tipo de misiones van aquí

- Hábitos que estoy formando o fortaleciendo
- Rutinas diarias o semanales que quiero mantener
- Procesos de crecimiento personal que estoy trabajando
- Sistemas de disciplina, salud, lectura, o lo que necesite
