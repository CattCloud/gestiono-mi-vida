# Evolución del sistema

Registro de ideas de mejora, problemas detectados y situaciones que el sistema aún no resuelve. Cada entrada nace de una situación real durante el uso del sistema.

Cuando el agente IA o yo detectemos algo que necesita mejora, se registra aquí con el contexto de lo que pasó. No se resuelve en el momento — se captura para no perderlo.

---

## Formato de entrada

```
### [fecha] Título breve

**Situación:** Qué estaba pasando cuando se detectó esto.
**Problema / Idea:** Qué le falta al sistema o qué se podría mejorar.
**Prioridad:** alta / media / baja
**Estado:** pendiente / en progreso / resuelto
```
