# Aclarador Diario

Aquí van mis reflexiones del día. No es obligatorio escribir todos los días — solo cuando sienta que necesito aclarar mi mente.

Un aclarador diario es un momento para pausar y escribir: qué hice, qué me falta, cómo me siento respecto a lo que estoy haciendo. Es mi forma de procesar el día.

## Formato

Cada archivo se nombra con la fecha: `2026-04-12.md`
