# Enter Tech School

Aquí vive todo lo relacionado con mi trabajo en el Bootcamp Enter Tech School.

Cada proyecto del trabajo es una **misión** (una carpeta con `_aclarador.md` adentro). Dentro de cada misión pueden existir **sub-misiones** (carpetas con su propio `_aclarador.md`) y **tareas** individuales (cada una es un `.md` con frontmatter rico). Las tareas que no pertenecen a ninguna sub-misión viven en `[mision]/tareas-sueltas/`.
