# Integración Kommo ↔ Enterbase

## Aclarador
Enter Tech School necesita comunicar el CRM **Kommo** con la App Enterbase. El objetivo definido en reunión con las chicas del equipo es lograr esa integración; la solución inicial que se propuso fue una API (https://developers.kommo.com/docs/private-integration) por ser lo primero que vino a mente, pero posiblemente existan otras formas (webhooks, exportación/importación, intermediarios, etc.).
