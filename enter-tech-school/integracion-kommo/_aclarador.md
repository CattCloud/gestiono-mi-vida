# 🎯 Integración Kommo ↔ Enterbase

## Aclarador

Enter Tech School necesita comunicar el CRM **Kommo** con la App Enterbase. El objetivo definido en reunión con el equipo es lograr esa integración; la solución inicial propuesta fue una API (https://developers.kommo.com/docs/private-integration) por ser lo primero que vino a mente, pero posiblemente existan otras formas (webhooks, exportación/importación, intermediarios, etc.).

Con la investigación se generará un documento con la propuesta del método de integración, que se expondrá en la reunión **"Coordinación Proyecto: API Kommo & EB"** el viernes 17-abr-2026 de 3:00 a 3:30 PM con Mhitzy.

**Prioridad:** dentro de Enterbase, esta integración va después de los descargables/exportaciones y del registro manual de asistencias/entregables.
